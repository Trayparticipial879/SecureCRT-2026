## ❓ Frequently Asked Questions

What should I do if the connection times out?
Check your internet connection and verify that the server name or IP address is correct. Ensure that the remote server allows external connections through the firewall.

Can I move my settings to another computer?
Yes. You can export your session database through the Options menu. Save the file and import it on the new installation to keep your configuration.

How do I generate SSH keys?
Go to the Tools menu and select Key Manager. Follow the instructions to create a public and private key pair. Use the public key on the remote server to grant access without a password.

Does this work on Windows 7?
This version supports Windows 10 and Windows 11. We do not provide testing or support for older versions of Windows.
